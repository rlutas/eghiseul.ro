Overview

Ce este eGhiseul.ro?

Platforma digitala care simplifica accesul la documente si servicii guvernamentale pentru cetatenii romani, in special cei din diaspora.

Platforma Actuala

  • WordPress + WPForms
  • Comenzi in Google Sheets
  • Contracte manuale
  • KYC problematic
  • Fara cos cumparaturi

Platforma Noua

  • Next.js + Node.js
  • Admin Dashboard centralizat
  • Contracte automate
  • KYC cu OCR + persistent
  • Cart + multi-serviciu

Ce e Nou in v2.0

OCR-First

Upload CI la pasul 1, extragem automat CNP, nume, adresa. Verificam daca actul e expirat.

NOU

KYC Persistent

KYC salvat 180 zile in cont. La comenzi noi, skip complet daca e valid.

NOU

Cart System

Adauga multiple servicii in cos. Checkout unic pentru tot. Pre-fill pentru aceeasi persoana.

NOU

Smart Pre-fill

Alt serviciu pentru aceeasi persoana = 0 campuri de completat, doar intrebari specifice.

NOU

PJ Bulk Orders

Firme comanda pentru N angajati. Template Excel. Discount volum 5-20%.

NOU

SMS Notificari

Optional +5 RON. SMS la fiecare schimbare status: plata, document gata, expediat, livrat.

NOU
