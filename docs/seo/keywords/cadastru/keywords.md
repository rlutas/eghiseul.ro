# Cadastru (releveu / plan de încadrare / plan cadastral) — Keyword Research

**Sursă:** Semrush keyword ideas, 2026-06-25 (capturi releveu / plan de încadrare / plan cadastral).
**Intent:** I = informațional, C = comercial/tranzacțional.
**Concluzie:** KD foarte mic (2–19) pe tot clusterul cadastral = winnable. Strategie mixtă:
pagini de serviciu pentru termenii tranzacționali + 2–3 articole pentru head-urile informaționale
cu volum mare („releveu", „plan cadastral", „ce este…").
