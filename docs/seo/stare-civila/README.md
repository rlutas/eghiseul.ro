## De ce topical, nu locație
~3.200 primării emit acte de stare civilă, iar biroul relevant e cel **unde s-a înregistrat actul**, nu orașul userului → o matrice pe orașe ar fi thin + înșelătoare. Deci cluster pe **sub-intenții** (duplicat, transcriere, valabilitate, acte necesare).
