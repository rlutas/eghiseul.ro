## COMMON MISTAKES TO AVOID

❌ **Don't:**
- Stuff keywords unnaturally (sounds robotic)
- Copy competitor content (not unique)
- Hide keywords in white text or tiny font
- Create duplicate meta tags across services
- Ignore mobile optimization
- Have broken links in content
- Forget ALT text on images
- Mix up schema types (Service vs Product)
- Over-link to irrelevant pages
- Ignore user experience for SEO

✓ **Do:**
- Write for humans first, search engines second
- Create original, detailed content
- Use keywords naturally in headings
- Optimize for voice search (long-tail)
- Test mobile first
- Check all links regularly
- Use semantic HTML (H1-H6 hierarchy)
- Verify schema markup periodically
- Link strategically to related content
- Monitor metrics continuously
