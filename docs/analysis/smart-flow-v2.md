# eGhiseul - Smart Flow v2.0

## Principii de design

### 1. OCR-First
- Upload CI/Pasaport **devreme** in flow
- Extrage automat: CNP, Nume, Adresa, Data nastere
- Verifica expirare act
- Pre-completeaza toate campurile

### 2. KYC o singura data
- Daca user logat cu KYC valid â†’ skip complet
- Daca a uploadat CI â†’ doar selfie necesar
- KYC salvat 180 zile in cont

### 3. Cart System
- Adauga multiple servicii in cos
- Pentru aceeasi persoana â†’ date pre-completate
- Pentru alta persoana â†’ doar date noi
- Checkout unic pentru tot cosul

### 4. Smart Pre-fill
- Aceeasi persoana = 0 campuri de completat
- Alt serviciu = doar intrebari specifice acelui serviciu
- Firma PJ = CUI â†’ auto-fill toate datele

### 5. SMS Notificari (optional +5 RON)
- Status updates automate
- AWB tracking
- Document gata
