## Responsive Breakpoints

### Mobile (< 640px)
- Container: 100% width, 16px side padding
- Font sizes: Reduce H1 to 24px
- Buttons: Full width, 44px height (touch-friendly)
- Trust badges: Stack vertically
- Spacing: Reduce large margins by 8px

- Container: Max 448px, centered
- All desktop styles apply
- Touch targets: Minimum 44px

### Desktop (> 1024px)
- Container: Max 448px, centered
- Touch targets: Minimum 32px
