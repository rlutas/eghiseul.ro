## Color Usage Guidelines

### Do's ✅
- Use Romanian blue (#003380) for all primary actions
- Use legal green (#0F7A3E) for success/verification
- Use Romanian gold (#F5B800) sparingly (< 5% of page)
- Ensure text contrast ≥ 4.5:1
- Pair colors with icons for accessibility

### Don'ts ❌
- Don't use Tailwind blue-600 (#2563eb) - use Romanian blue instead
- Don't use bright flag yellow (#FCD116) as background
- Don't use flag red (#CE1126) as primary (too bright)
- Don't rely on color alone to convey meaning
- Don't overuse gold accent (dilutes impact)
